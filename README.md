# База данных Шипкидов

Веб-сайт с базой данных вымышленных персонажей (шипкидов) из вселенных Undertale.

## Сайт
Доступен по адресу: https://shipkidsau.github.io

## Структура
- `index.html` - главная страница с фильтрацией
- `faq.html` - страница с вопросами и ответами
- `style.css` - стили сайта
- `script.js` - логика фильтрации и поиска
- `images/` - папка с изображениями персонажей

## Функционал
- Поиск по именам шипкидов
- Фильтрация по родителям (до 2 одновременно)
- Адаптивный дизайн для мобильных
- Темная тема

## Лицензия
Все права на персонажей принадлежат их владельцам.
